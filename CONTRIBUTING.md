# Contributing

Thanks for contributing to **node-gen-kit**.

## Setup

```bash
npm install
npm test
npm run typecheck
npm run build
```

## Guidelines

- TypeScript strict mode
- Keep the plugin interface stable
- Add tests for new generators / plugins
- Do not commit secrets
- Do not add AI co-author trailers, AI tool signatures, or "generated by …" attribution in commits or files

## Pull requests

1. Fork and branch from `main`
2. Keep changes focused
3. Ensure CI is green
4. Describe what and why in the PR

## Code of conduct

Be respectful. See [CODE_OF_CONDUCT.md](./CODE_OF_CONDUCT.md).
